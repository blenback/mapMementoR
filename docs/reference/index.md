# Package index

## All functions

- [`choose_relief_colors()`](choose_relief_colors.md) : Choose relief
  colors for hillshade based on map background color
- [`create_race_map()`](create_race_map.md) : Create a map of a race
  route with OSM background and elevation chart
- [`create_race_series()`](create_race_series.md) : Loop over multiple
  races and styles to create maps
- [`geom_relief()`](geom_relief.md) : Hillshade and shaded relief
  functions for ggplot2
- [`get_hillshade()`](get_hillshade.md) : Get hillshade elevation data
  for a bounding box
- [`get_osm_components()`](get_osm_components.md) : Get OSM map
  components (highways, streets, water, sea) for a bounding box and
  location
- [`get_page_dimensions()`](get_page_dimensions.md) : Returns width and
  height in mm for common page sizes
- [`get_track_stats()`](get_track_stats.md) : Get max/min distance and
  elevation from multiple GPX files
- [`parse_gpx()`](parse_gpx.md) : Parse GPX file and extract track
  points
- [`save_powerof10_to_yaml()`](save_powerof10_to_yaml.md) : Save
  Powerof10 athlete race data to YAML
