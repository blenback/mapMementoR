# Returns width and height in mm for common page sizes

Returns width and height in mm for common page sizes

## Usage

``` r
get_page_dimensions(page_size, orientation = "portrait")
```

## Arguments

- page_size:

  Page size string (e.g., "A5", "A4")

- orientation:

  Page orientation: "portrait" (default) or "landscape"

## Value

List with width and height
